# centric-linkedIn
implement frontend using doar react theme  : [Doar React theme](https://doar-react.netlify.app/)

### User 16.0.0 node version

# Installation

```
# install packages using yarn 
$ yarn install 

# install lerna
$ npm install -g lerna

# install packages
$ yarn bootstrap
```

# Run Server

```
# run server
$ yarn start:main
```
